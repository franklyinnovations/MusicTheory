Welcome to the Keyboard. This is meant as a little practice keyboard to play around with when you don't have a piano nearby.

To use this, you need to click on the keyboard above and then use the keys on your computer keyboard to play notes. You can select a key and mode to view a scale. You can select 'Limit to Key' to only let notes get played within that scale.

Sorry mobile users, this only works for desktops/laptops right now.
